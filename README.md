# Lead-quality-performance-and-Optimization-Dashboard
Lead Quality Performance & Optimization Dashboard

This project presents an interactive Power BI dashboard designed to analyze and optimize lead quality performance. The dashboard evaluates lead conversion metrics, quality rates, and revenue impact across multiple dimensions to support data-driven decision-making.

🔹 Key Features:

Total Leads, Good Leads, Lead Quality Rate, and Closed Rate KPIs

Revenue Impact Analysis

Lead Quality Rate (LQR) breakdown by:

Publisher Campaign

Debt Level

Address Score Group

Phone Score

Partner

Widget Name

Ad Group

Branded vs Generic

Monthly and Yearly trend analysis

Interactive slicers (Search vs Content, Debt Level, Address Score, Widget Name)

Performance comparison against target quality benchmarks

🛠 Tools Used:

Power BI

Power Query (Data Cleaning & Transformation)

DAX (Calculated Measures & KPIs)

This dashboard helps identify high-performing segments, eliminate poor-quality leads, and optimize revenue impact through strategic filtering and performance tracking.
